The Kontalk Messaging Project

A new way of communicating.
What is it
Kontalk is an effort to write an instant messaging system – primarly designed for embedded devices, especially smartphones.
It would be cross-platform, though there is only an Android client for now. iOS, Symbian and even a desktop client will be coming soon.
The network
Kontalk is not owned by anyone. The network is run by the community for the community, in a network of servers that can expand virtually to infinity.
We believe communication is a human right; therefore, anyone on the planet should be able to connect freely and for free. Kontalk is designed to be free and to remain as such forever.
Distributed network
The Kontalk network can continue running even if just one of the servers is up. The network itself is designed to not allow any single point of failure and it protects itself from broken servers.
Security
All data exchanged inside the Kontalk network is encrypted, and user IDs are always hashed, so that even the server owners cannot retrieve the phone number of a user.
Lightweight protocol
Kontalk protocol has been designed from scratch to be very lightweight and to consume very low network traffic in client-to-server communication. All the specifications are available at this page.
Web of Trust
The Kontalk network of servers is public: anyone can join with their own server. A server is untrusted at the beginning; by means of a vote system – based on strong asymmetric encryption – and by working actively in the network it can become a trusted server and get more privileges.
