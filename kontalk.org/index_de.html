Verschlüsselt

Immer verschlüsselt. Sowohl die Kanäle vom Client zum Server als auch von Server zu Server, sind komplett verschlüsselt. Benutzer können durch Ende-zu-Ende Verschlüsselung sicher sein — die Server können nichts entschlüsseln.

Verteilt

Betrieben von der Community für die Community. Kontalk wird von einer Gemeinschaft von Freiwilligen betrieben, die Server anbieten und die Kosten unter sich aufteilen. Zur Netzwerkseite gehen (englisch) »

Open source

Jedes einzelne Bit der Software welches im Kontalknetzwerk verwendet wird, sowohl Server wie auch Client, ist völlig quelloffen. Einsehbar bei Github.

  • Kostenlos. Für immer.

    Kontalk basiert auf einem, von Freiwilligen betrieben Netzwerk von Servern, um einen stabilen und sicheren Dienst für alle Kontalknutzer zur Verfügung zu stellen. Aus diesem Grund wird Kontalk für immer kostenlos bleiben.

  • Perfekt für's Smartphone.

    Kontalk verwendet Ihre Telefonnummer, um Sie bei Ihren Kontakten zu identifizieren, ohne dass sie sich einen Benutzernamen oder Nutzer IDs merken müssen.

  • XMPP basierend.

    Kontalk basiert auf felsenfesten Internetstandards, welche die Zwischenfunktionsfähigkeit mit anderen Diensten sicherstellt.

    * Die XMPP Version ist nur als Beta verfügbar.
