  • What is Kontalk

    Kontalk is an effort to write an instant messaging system - primarly designed for embedded devices, especially smartphones. Though currently there is only an Android Client, it aims to be cross-platform compatible. iOS, Symbian and even a desktop client will be developed (developers wanted!).

  • Web of Trust

    The Kontalk network of servers is public: anyone can join with their own server. A server is untrusted at the beginning. By means of a voting system - based on strong asymmetric encryption - and by working actively in the network, it can become a trusted server and get more privileges.
